#### Installation

```bash
npm install lenis/react
```

We use lenis for smooth scrolling:

```tsx
'use client'
import { ReactLenis } from 'lenis/react'
export default function HorizontalScroll(): JSX.Element {
  return <ReactLenis root>{/* content */}</ReactLenis>
}
```
